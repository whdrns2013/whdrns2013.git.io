# 주의
SQL 상황이 아닌, 바깥 terminal에서 실행할 수 있다.


# DB export (dump)

데이터베이스 export
```terminal
mysqldump -u root -p DB명 > 저장할경로.sql
```

특정 테이블만 제외하고 export
```terminal
mysqldump -u root -p --ignore-table=테이블명 DB명 > 저장할경로.sql
```

# DB import

```terminal
mysql -u root -p DB명 < 파일경로.sql
```
