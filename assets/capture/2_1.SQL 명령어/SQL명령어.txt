■■■ 복합키
https://javacpro.tistory.com/11
- 두 개 이상의 칼럼의 조합을 통해 PRIMARY KEY를 만드는 방법
- 유일성을 만족하지 못할 경우 이렇게 하는 듯 하다.

CREATE TABLE 테이블명 (
	칼럼명1 데이터타입 제약조건,
	칼럼명2 데이터타입 제약조건,
	칼럼명3 데이터타입 제약조건,
	PRIMARY KEY ('칼럼명1', '칼럼명2')
	);

아래는 잘못된 사용법
복합키는 복합키를 만드는 것이지, 두 칼럼을 PRIMARY 키로 만드는 것이 아니다.

CREATE TABLE 테이블명 (
	칼럼명1 데이터타입 제약조건 PRIMARY KEY,
	칼럼명2 데이터타입 제약조건 PRIMARY KEY,
	칼럼명3 데이터타입 제약조건,
	);

■■■ CASE, WHEN, THEN
https://extbrain.tistory.com/46
언어에서 스위치문과 비슷
CASE
	WHEN 조건
	THEN '반환 값'
	WHEN 조건
	THEN '반환 값'
	ELSE 'WHEN 조건에 해당 안되는 경우 반환 값'
END
- WHEN과 THEN은 한쌍이어야 합니다.
- WHEN과 THEN은 다수가 존재할 수 있습니다.
- ELSE가 존재하면 모든 조건에 해당하지 않는 경우에 반환 값을 설정할 수 있습니다.
- ELSE가 존재하지 않고, 조건에 맞지 않아서 반환 값이 없으면 NULL를 반환합니다.
- CASE 문은 END로 종료
